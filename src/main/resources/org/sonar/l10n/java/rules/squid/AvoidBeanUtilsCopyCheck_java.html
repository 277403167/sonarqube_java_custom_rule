避免用Apache Beanutils进行属性的copy

只有当两个类的属性相同才会进行处理，对不同属性的不做处理；对java.util.Date类不支持，但是对java.sql.Date支持；对Integer,Boolean,Long类型不支持， 默认值为null，但是使用方法后默认值为0，所以要小心使用。使用的效率也比较低，所以要谨慎使用。

Noncompliant Code Example

	public void getcopyProperties(Object stuVo, Object stu) throws Exception {
		BeanUtils.copyProperties(stuVo, stu); // Noncompliant
	}
